**Sandboxer 0.2.3**

Sandboxer is a Crystal shard that runs shell commands inside a platform-native sandbox with a configurable access policy. It works on macOS (via `sandbox-exec` / Seatbelt) and Linux (via `bwrap` / Bubblewrap), presenting both as a single clean API.

**Changes**

- **Python preset support** — `Sandboxer::Preset::Python` adds static policies for Homebrew (macOS ARM/Intel, Linux) and system Python (Linux), plus a `for_executable(path)` builder for version-managed interpreters (`pyenv`, `uv python install`).
- **Virtual environment support** — `Preset::Python.for_venv(path)` derives a policy from a project's `.venv`, resolving its base interpreter automatically so you don't need to specify both.
- **New CLI flags** — `--python PATH` for a specific interpreter, `--python-venv PATH` for a project virtualenv, e.g. `sandboxer run --python-venv .venv -- python3 script.py`.
