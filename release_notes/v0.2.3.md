**Sandboxer 0.2.3**

Sandboxer is a Crystal shard that runs shell commands inside a platform-native sandbox with a configurable access policy. It works on macOS (via `sandbox-exec` / Seatbelt) and Linux (via `bwrap` / Bubblewrap), presenting both as a single clean API.
