**Sandboxer 0.2.2**

Sandboxer is a Crystal shard that runs shell commands inside a platform-native sandbox with a configurable access policy. It works on macOS (via `sandbox-exec` / Seatbelt) and Linux (via `bwrap` / Bubblewrap), presenting both as a single clean API.

**Changes**

- **Ruby preset support** — `Sandboxer::Preset::Ruby` adds static policies for Homebrew (macOS ARM/Intel, Linux) and system Ruby (Linux), plus a `for_executable(path)` builder for version-managed installs (`rbenv`, `chruby`, `asdf`, `ruby-install`) where the install location isn't fixed.
- **New `--ruby PATH` CLI flag** — `sandboxer run --ruby $(which ruby) -- ruby script.rb` derives a policy from whichever Ruby is active in your shell, including version managers.
- **More flexible preset dispatch** — presets are no longer required to support every platform uniformly, enabling presets like Ruby's Linux-only system layout.
